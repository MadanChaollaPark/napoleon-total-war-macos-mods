# Ottoman naval parity

This standalone Mod-type pack gives the WW0 Ottoman faction access to five
generic late-game naval units after the normal building and technology
requirements are met:

- Carronade Frigate
- Steam Frigate
- 38-gun Steam Ship
- 80-gun Steam Ship
- Ironclad

It does not grant technology, buildings, money, or units for free. WW0 already
contains the drydock, research requirements, models, UI, and Ottoman naval-crew
mapping; the missing final faction-permission rows are the only change here.

- `WW0_Ottoman_Naval_Parity.pack`

# Middle Eastern agent parity

Napoleon uses different agent classes for Middle Eastern cultures:

| European class | Middle Eastern class |
|---|---|
| Gentleman | Eastern Scholar |
| Rake | Assassin |

Vanilla and WW0 connect education and secret-building benefits to the European
classes but leave the equivalent Eastern effect keys unused. This standalone
pack mirrors the normal benefits without removing or weakening the original
European rows.

It adds:

- Scholar spawn rates of 4 / 8 / 16 at College / University / Enlightened University.
- Scholar factionwide caps of 1 / 2 / 3 from those buildings.
- Assassin spawn rates of 4 / 8 / 16 at Gentlemen's Club / Secret Society / Masonic Lodge.
- Assassin factionwide caps of 1 / 2 / 3 from those buildings.
- The equivalent +1 cap from Public Schooling and Passports research.

The pack applies through culture-aware agent definitions, covering WW0's
Ottomans and Crimean Khanate. The separate startpos component corrects their
base cap records for newly started campaigns.

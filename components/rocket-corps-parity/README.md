# WW0 Rocket Corps parity

The Rockets technology has no passive benefit, while the base game restricts
all Rocket Troops to five original European majors. This standalone pack adds:

- A European-minor Rocket Troop.
- An Eastern Rocket Troop for the Ottoman Empire and Crimea.

Both require `military5_rockets` and the Engineer School. Every faction reuses
its own existing artillery appearance and exact colour record. The AI quality
is the same `800` value used by the five original Rocket Troops.

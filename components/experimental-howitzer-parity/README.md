# WW0 Experimental Howitzer parity

Only the five original European majors receive a unit from Carcass Shot in the
base database. This standalone pack adds two shared variants for the other 36
WW0-playable factions:

- A European-minor Experimental Howitzer.
- An Eastern Experimental Howitzer for the Ottoman Empire and Crimea.

Both require `military3_carcass_shot` and the normal Ordnance Board or Engineer
School. Every faction uses its own existing artillery appearance and colour
record. The AI quality is the same `200` value used by the five original
Experimental Howitzers.

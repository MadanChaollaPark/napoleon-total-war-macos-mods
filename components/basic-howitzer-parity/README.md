# WW0 basic Howitzer parity

This standalone pack gives Spain, Portugal, Switzerland and Crimea access to
the existing generic 7-lber Howitzer from the normal Great Arsenal, Ordnance
Board and Engineer School chain.

Each faction receives a new, uniquely keyed uniform mapping that reuses its own
existing artillery appearance and exact RGB colour record. Donor uniform keys
are not overwritten.

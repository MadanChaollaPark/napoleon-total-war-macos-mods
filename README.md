# Napoleon: Total War macOS Mods

Modular, reversible fixes for the Feral macOS release of Napoleon: Total War.

The repository is intentionally component-based: players can install only the
faction unlocker, Ottoman naval parity, Middle Eastern agent parity, or the WW0
new-campaign agent-cap correction.

> Work in progress: installable components and verification tooling are added
> in separate commits so each feature remains independently reviewable.
